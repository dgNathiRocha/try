Horizontal Histogram

In statistics, a histogram is representation of the distribution of numerical data, where the data are binned and the count for each bin is represented. More generally, in Plotly a histogram is an aggregated bar chart, with several possible aggregation functions (e.g. sum, average, count...) which can be used to visualize data on categorical and date axes as well as linear axes. Alternatives to histogram plots for visualizing distributions include violin plots, box plots, ECDF plots and strip charts.
