Bubble Chart

A multi-variable graph that combines elements of a proportional area chart and a scatterplot is called a bubble chart. Similar to a scatterplot, a bubble chart plots points along a grid where the X and Y axes are independent variables using a Cartesian coordinate system. In contrast to a scatterplot, each point has a label or category given to it, which can be shown next to the point or on a legend. Afterwards, the area of each plotted point's circle represents a third variable. Additionally, colors can be used to indicate additional data variables or to differentiate across groups. Time can be displayed by animating the data variables to change over time, or by having it as a variable on one of the axes.
